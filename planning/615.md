 ## 数据库的安装与简单实用（分享给侬们大家呀）
### 运行MySQL   
先用命令mysql -u root -p进入MySQL    
再用命令create database stu 创建了数据库stu    
### 创建数据库  
命令：create database <数据库名>  
 eg: mysql> create database stu;  
### 显示数据库   
命令：show databases  
### 删除数据库  
命令：drop database <数据库名>   
 eg: mysql> drop database stu;  
### 使用数据库
命令： use <数据库名>
