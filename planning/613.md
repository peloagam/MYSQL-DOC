# Learning and Knowing
 1. 安装GIT  
sudo apt-get install git  
 2. 创建目录  
 mkdir github
 cd github  
 3. 创建git仓库.  
    git init
 4. 文件  
- 创建文件readme  
 touch README  
- 编辑文件  
 vim README  
- 查看目录下文件的状态.  
  git status
- 把文件加到仓库中去 
 git add README
 5. 查看
- 查看目录下文件的状态.  
 git status
- 提交  
 git commit  
 - 查看提交信息  
  git log
