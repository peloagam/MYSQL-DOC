## Today I am strong!!
1. 安装GIT  
sudo apt-get install git  
2. 创建版本库
- 创建目录  
 mkdir github
 cd github  
-  使用git init命令, 将当前目录创建成git仓库.  
git init
3. 增加文件  
- 创建文件readme  
 touch README  
- 编辑文件
vim README  
- 查看当前状态 
 git status
- 把文件加到仓库中去.  
 git add README
- 提交  
 git commit  
4. 查看提交信息  
 git log
